# estudo-algoritmos

Versão do python usada: 3.11.7

Para executar o código insertionSort e selectionSort, basta clonar o repositório, acessar a pasta com o terminal e usar o seguinte comando:
python insertionSelectionSort.py

Para executar o código mergeSort e quickSort, basta clonar o repositório, acessar a pasta com o terminal e usar o seguinte comando:
python mergeSortQuickSort.py
